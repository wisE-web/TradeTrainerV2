1. **Install Node.js**: Download from https://nodejs.org/
2. **Open terminal** in this folder and run:
   ```bash
   npm install
   ```
3. **Create `.env` file** with your OpenAI API key:
   ```
   OPENAI_API_KEY=your_api_key_here
   ```
4. **Start the app**:
   ```bash
   npm start
   ```
5. **Open browser** to `http://localhost:5000`
